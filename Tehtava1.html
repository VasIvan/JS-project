JAVASCRIPT



Syntaksi

JavaScript-syntaksi tarkoittaa sääntöjä, jotka määrittävät, miten kieli kirjoitetaan (ohjelmoija) ja tulkitaan (selaimen avulla).

Esimerkiksi:

<script>
document.write("JavaScript is not Java");
</script>

Muuttujat

Muuttuja tarkoittaa mitä tahansa, joka voi vaihdella. JavaScript sisältää muuttujia, joilla on datan arvo ja jota voidaan muuttaa milloin tahansa.
JavaScript käyttää varattua avainsanaa var ilmoittaa muuttuja. Muuttujalla on oltava yksilöllinen nimi. Voit määrittää arvon muuttujalle käyttäen yhtä suurta (=) operaattoria, kun ilmoitat sen tai ennen sen käyttöä.

Syntaksi:

var (variable-name); var (variable-name) = (value);

Esimerkiksi:

var one = 1; // variable stores numeric value

var two = 'two'; // variable stores string value

var three; // declared a variable without assigning a value

Kommentit

Koodin sisällä olevat selittävät huomautukset. Kommentteja käytetään muistuttamaan itseäsi ja ilmoittamaan muille ohjelman toiminnasta. Moniarvoisia kommentteja käytetään koodin suuriin tekstikuvauksiin tai kommentoida koodikappaleita virheenkorjausohjelmia käytettäessä. Yhden rivin kommentit merkitään kahdella eteenpäin viivalla. Kääntäjä ei huomioi kommentteja.

Syntaksi:

Moniarvoisia kommentteja - /* */

Yhden rivin kommentit - //

Esimerkiksi:

var one = 1; // kommentti täällä ....
