LASKENTA- JA VERTAILUOPERAATTORIT



Vertailuoperaattorit

Operaattorit, jotka vertaavat arvoja ja palaavat totta tai vääriä. Operaattorit sisältävät:>, <,> =, <=, === ja !==

  • Alle (<)

    Palautuu totta, jos vasemmalla oleva arvo on pienempi kuin oikealla oleva arvo, muuten se palauttaa väärän.

    Esimerkiksi:

    var x = 5;
    x < 8 (true);

  • Suurempi kuin (>)

    Palaa totta, jos vasemmanpuoleinen arvo on suurempi kuin oikealla oleva arvo, muuten se palauttaa väärän.

    Esimerkiksi:

    var x = 5;
    x > 8 (false);

  • Alle tai yhtä suuri (<=)

    Palautuu totta, jos vasemmalla oleva arvo on pienempi tai yhtä suuri kuin oikealla oleva arvo, muuten se palauttaa väärän.

    Esimerkiksi:

    var x = 5;
    x <= 8 (true);

  • Suurempi tai yhtä suuri (> =)

    Palautuu totta, jos vasemmanpuoleinen arvo on suurempi tai yhtä suuri kuin oikealla oleva arvo, muuten se palauttaa väärän.

    Esimerkiksi:

    var x = 5;
    x >= 8 (false);

  • Yhtä suuri (===)

    Palaa totta, jos vasemmanpuoleinen arvo on yhtä suuri kuin oikealla oleva arvo, muuten se palauttaa väärän.

    Esimerkiksi:

    var x = 5;
    x === 8 (false);
    x === 5 (true);
    x === "5" (false);

  • Ei yhtäläinen (! ==)

    Palaa totta, jos vasemmalla oleva arvo ei ole sama kuin oikealla oleva arvo, muuten se palauttaa väärän.

    Esimerkiksi:

    var x = 5;
    x !== 8 (true);
    x !== 5 (false);
    x !== "5" (true);

Loogiset operaattorit

Operaattorit, jotka yhdistävät useita boolen ilmaisuja tai arvoja ja tarjoavat yhden boolen ulostulon. Operaattoreita ovat: &&, || ja ! .


  • && (ja)

    Tämä operaattori on totuus (toimi kuten totta), jos ja vain, jos ilmaisut molemmilla puolilla ovat totta.

    Esimerkiksi:

    var x = 6;
    var y = 3;
    (x < 10 && y > 1) - true;

  • || (tai)

    Tämä operaattori on totuus, jos sen molemmilla puolilla oleva ilmaisu on totta. Muuten se on väärä (toimi kuin väärä).

    Esimerkiksi:

    var x = 6;
    var y = 3;
    (x == 5 || y == 5) - false;
