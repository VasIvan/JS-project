TIETOTYYPIT



JavaScriptissä on kaksi erilaista tietoa: primitiivit ja esineet. Primitiivinen on yksinkertaisesti tietotyyppi, joka ei ole objekti, eikä sillä ole menetelmiä.

Primitiivit

JS: ssä on kuusi primitiivistä tietotyyppiä:

  1. Boolean

    Boolean edustaa vain yhtä kahdesta arvosta: true tai false.
    Ajattele boolean kuin on / off tai yes / no-kytkin.

    Esimerkiksi:

    var boo1 = true;
    var boo2 = false;

  2. Number

    JavaScriptissä on vain yksi tyyppi. Numerot voidaan kirjoittaa desimaalipisteellä tai ilman sitä.
    Numero voi olla myös + Infinity, -Infinity ja NaN (ei numero).

    Esimerkiksi:

    var num1 = 32;
    var num2 = +Infinity;

  3. String

    Merkkijonoja(Strings) käytetään tekstin tallentamiseen. Merkkijonojen(Stringsin) on oltava joko kaksinkertaisia tai yksittäisiä lainausmerkkejä. JS: ssä merkkijonot(stringsit) ovat muuttumattomia (niitä ei voi muuttaa).

    Esimerkiksi:

    var str1 = 'hello, it is me';
    var str2 = "hello, it's me";

  4. Null

    Merkkijonoja(Strings) käytetään tekstin tallentamiseen. Merkkijonojen(Stringsin) on oltava joko kaksinkertaisia tai yksittäisiä lainausmerkkejä. JS: ssä merkkijonot(stringsit) ovat muuttumattomia (niitä ei voi muuttaa).

    Esimerkiksi:

    var testVar;
    console.log(testVar); // undefined

  5. Undefined

    'Undefined' tietotyypillä voi olla vain yksi arvo - erityinen arvo 'undefined'. Jos muuttuja on ilmoitettu, mutta sille ei ole annettu arvoa, sillä on arvo "undefined".

    Esimerkiksi:

    var str1 = 'hello, it is me';
    var str2 = "hello, it's me";

  6. Symbol

    Symboli on ainutlaatuinen ja muuttumaton tietotyyppi ja sitä voidaan käyttää tunnisteena objektin ominaisuuksiin. Symboliobjekti on symboli-primitiivisen datatyypin implisiittinen objektikääre.

    Esimerkiksi:

    const mySymbol = Symbol('mySymbol');

Objectit

Objekti on järjestämätön luettelo primitiivisistä tietotyypeistä (ja joskus viitetiedostotyypeistä), jotka tallennetaan sarja-arvopareiksi. Kutakin luettelon kohdetta kutsutaan ominaisuudeksi (toimintoja kutsutaan menetelmiksi).

Esimerkiksi:

var myFirstObject = {firstName: "Richard", favoriteAuthor: "Conrad"};
